# 知识星球数据总表使用规则提炼

## 三个核心系统

- 趋势识别：日、周、月三个级别分别给出上行、下行、无趋势，并记录当前状态持续时间。三周期组合可理解为 27 种状态空间。
- 相对比价：关注 `lead`、`Improving`、`Weakening`、`Lag` 四类状态，以及相对强度、强度动量、早期转折三条分数。早期转折最敏感，强度动量次之，相对强度最终反映到价格表现。
- 资金监控：`加杠杆` 表示资金行为偏多，`去杠杆` 表示资金行为偏空；持续时间、状态涨幅、资金数值和前日变化用于判断资金行为是否延续。

## 默认观察框架

- 重点观察：比价状态为 `lead` 或 `Improving`，资金状态为 `加杠杆`，周级别趋势为 `上行趋势`。
- 风险观察：比价状态为 `Lag` 或 `Weakening`，资金状态为 `去杠杆`，周级别趋势为 `下行趋势`。
- 状态切换：比价状态持续时间为 1 或资金状态持续时间为 1 时，视为新切换，进入日报的状态变化区。

## 解读原则

- 绝对趋势、相对比价、资金行为需要交叉验证，单一指标不独立构成结论。
- 资金监控有效性的核心假设是大资金行为具有持续性，但环境变化可能导致历史有效性下降。
- 星球原始材料明确强调所有内容仅供参考；本项目输出延续同一边界，不给买卖建议。

## 后续扩展

- 补充更多历史日报后，可以统计每个资产的状态转移路径、资金状态持续收益、状态切换后的表现分布。
- 接入行情后，可以把指标曲线与价格曲线按日对齐，观察信号领先或滞后的情况。
