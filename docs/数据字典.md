# 数据字典

## 数据表

- `assets`：资产主表，使用 `asset_key = 代码|名称` 作为主键，避免同代码不同名称的期货品种互相覆盖。
- `observations`：每日星球指标表，使用 `dataset_date + dataset_type + source_row_number` 保留原始表每一行。
- `import_logs`：导入日志，使用文件 SHA256 去重。
- `price_bars`：行情缓存表，按资产、日期、来源存储 OHLCV。
- `symbol_map`：资产代码到免费行情源 symbol 的映射表。

## 标准字段

| 字段 | 含义 |
| --- | --- |
| `asset_code` | 原始代码 |
| `asset_name` | 标的名称 |
| `day_trend` / `week_trend` / `month_trend` | 日、周、月趋势状态 |
| `*_trend_duration` | 对应趋势持续时间 |
| `close_position_60d` | 收盘价相对 60 日位置 |
| `relative_strength` | 相对强度 |
| `strength_momentum` | 强度动量 |
| `early_turn` | 早期转折 |
| `relative_state` | 当前比价状态 |
| `relative_state_duration` | 当前比价状态持续时间 |
| `relative_state_return` | 当前比价状态涨幅 |
| `capital_state` | 当前杠杆资金状态 |
| `capital_state_duration` | 当前杠杆资金状态持续时间 |
| `capital_state_return` | 当前杠杆资金状态涨幅 |
| `capital_value` | 杠杆资金数值 |
| `capital_daily_change` | 杠杆资金相比前日变动 |

## 派生字段

| 字段 | 含义 |
| --- | --- |
| `trade_decision.action` | `long`、`short`、`wait`，分别对应可做多、可做空、不做/观望 |
| `quadrant.x` | 四象限横轴坐标，等于 `relative_strength - 100` |
| `quadrant.y` | 四象限纵轴坐标，等于 `strength_momentum - 100` |
| `quadrant.quadrant` | `Leading`、`Improving`、`Weakening`、`Lagging` |
